# 概要
私、ととが製作した学習教材の一部です。  

これからDApp開発を学習される方々向けに、有償コースで使用したソースコードを無料提供します。  
※DApp（分散アプリケーション）やスマートコントラクト開発における知識、技術力向上が主な目的です。私は日本におけるこれらの技術発展を望む者です。

コードの解説については現在動画（現時点では有料）のみで行なっています。もし関心のある方いらっしゃればご覧ください。
